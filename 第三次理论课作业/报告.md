
## 一、实验概述

### 1.1 实验目的

本实验旨在通过理论推导、边界延拓分析与数值实验，系统研究离散傅里叶变换（DFT）与离散余弦变换（DCT-II）之间的关系。具体目标包括：

- 明确DFT与DCT-II的数学定义、基函数特性及频域分解的共性原理；
- 从信号延拓角度阐明DCT作为偶对称边界条件下傅里叶分析的特例，解释DCT仅含余弦项而无虚部的原因；
- 对比两种变换隐含的边界延拓方式及其对信号连续性和频谱分布的影响；
- 通过实际图像数据量化比较DFT与DCT的能量集中性，验证DCT在图像压缩中的优势机理。

### 1.2 实验环境
- **软件环境**：Python 3.10，WSL Ubuntu 22.04.5 LTS，VS Code
- **依赖库**：numpy 1.24.3，opencv-python 4.8.0.76，matplotlib 3.7.1
- **测试数据**：标准灰度图像 Lena（分辨率 512×512），提取图像中间一行像素作为一维测试信号，信号长度 \(N=512\)

### 1.3 实验原理概述

DFT与DCT均为离散信号的正交变换，核心思想是将时域/空域信号分解到频域。DFT基于复指数基函数，隐含周期延拓边界条件；DCT-II基于实余弦基函数，隐含偶对称延拓边界条件。从信号展开角度看，DCT可视为对原信号进行偶对称延拓后再执行DFT，并利用对称性消除虚部的结果，因此是傅里叶分析在偶对称边界条件下的特例。两种变换均满足Parseval能量守恒定理，通过频域系数可实现信号的滤波、压缩等操作。

## 二、理论基础（含手写推导见附件）
### 2.1 DFT与DCT关键特性对比
（1）两者的基函数分别是什么
DFT的基函数：复指数函数，即正弦波与余弦波的复数组合。每个基函数同时包含余弦（实部）和正弦（虚部）成分，频率均匀分布在单位圆上。

DCT‑II的基函数：实余弦函数，只包含余弦波形，没有正弦成分。不同频率的余弦波在半个采样周期偏移下保持正交性。

（2）两者输出系数的主要形式有何不同
DFT的输出系数：一般为复数，包含实部和虚部。对于实信号（如图像像素值），系数满足共轭对称性，即正频率和负频率的系数互为共轭，存在信息冗余。

DCT的输出系数：纯实数，没有虚部，也不存在共轭对称冗余。每个系数直接表示对应频率余弦分量的强度，形式更简洁。

（3）DCT与DFT在“频域分解”思想上的共同点
两者都采用相同的核心思想：将原始信号表示为一组不同频率的基函数的线性组合，组合系数反映了信号在各个频率上的能量分布。具体共同点包括：

都是正交变换，基函数之间相互正交，能够唯一分解信号；

都满足能量守恒（Parseval定理），即变换前后信号的总能量不变；

都可以通过保留低频系数、舍弃高频系数来实现信号压缩或滤波。
### 2.1 DCT与DFT的关系推导

本部分通过手写推导完整展示DCT如何从偶对称延拓后的DFT导出，并回答以下三个问题：
1. DCT是否可看作某种特殊条件下的傅里叶分析？
2. DCT与偶对称延拓有何关系？
3. 为什么DCT中只出现余弦项，没有正弦项或显式虚部？



**推导结论**：
- DCT的本质是对原信号进行偶对称延拓（长度 \(2N\)）后，再对该延拓序列执行DFT。由于偶对称性，DFT的虚部（正弦分量）完全抵消，仅保留实部（余弦分量）。因此DCT是傅里叶分析在偶对称边界条件下的特例。
- DCT只含余弦项而无正弦项的根本原因是偶对称延拓导致正弦求和项正负抵消。
- 偶对称延拓保证了边界处的连续性（甚至一阶导数为零），这是DCT能量集中特性的数学根源。

## 三、实验结果与分析

### 3.1 实验数据与运行结果

实验采用标准Lena灰度图（512×512），提取第256行像素作为一维测试信号。程序成功读取图像后，信号长度 \(N=512\)，前10%系数个数 \(topk = \lfloor 512 \times 0.1 \rfloor = 51\)。能量占比计算结果如下：

| 变换 | 前51个系数能量占比 |
|------|-------------------|
| DFT  | 94.41%            |
| DCT  | 99.07%            |

### 3.2 延拓方式对比

实验生成的结果图 `dft_dct_result.png` 包含四个子图，分别展示了原始信号、DFT周期延拓、DCT偶对称延拓及频谱对比。




**观察结果**：
- **DFT周期延拓**：在 \(n=N\) 和 \(n=2N\) 的边界处，信号从末端值直接跳变回起始值。Lena图像行信号的首尾灰度值存在差异（左端点约130，右端点约110），因此跳变明显。
- **DCT偶对称延拓**：通过左右镜像拼接，边界处信号值自动连续（原首值与自身镜像值相等，原末值与自身镜像值相等），图中未见任何不连续点。

**分析**：DFT隐含的周期延拓假设要求信号首尾相等，但自然图像的一行像素通常不满足此条件，强行周期化会在边界处引入人为的高频成分。DCT的偶对称延拓则自然保证了边界连续，避免了额外高频分量的产生。

### 3.3 频谱与能量集中性分析

**能量占比结果**：DCT前51个系数占总能量的99.07%，而DFT仅占94.41%。DCT的能量集中性显著优于DFT。

**频谱特征**：
- DFT幅度谱：从低频开始下降，但在较高频段仍保留不可忽略的幅值，频谱曲线下降平缓。
- DCT系数：前几个系数占绝对主导，后续系数快速衰减至接近零。

**能量集中性差异的原因**：
1. **边界条件影响**：DFT的周期延拓在信号首尾不相等时引入突变，该突变对应广谱高频分量，迫使能量分散到多个频点。DCT的偶对称延拓使边界处信号连续且导数为零，不引入额外高频成分。
2. **基函数匹配性**：自然图像的灰度变化具有低通特性，DCT的实余弦基函数与此特性高度匹配；DFT的复指数基函数包含正负对称频率分量，对于实信号存在冗余，能量被“稀释”。
3. **冗余性差异**：DFT系数满足共轭对称性，实信号的完整信息仅需一半系数，但能量统计时正负频率分量均计入，导致低频能量占比偏低。DCT系数无此冗余，能量集中度更高。

**边界条件与能量集中性的关联**：周期延拓 → 边界跳变 → 高频分量增加 → 能量分散；偶对称延拓 → 边界连续 → 高频分量消除 → 能量高度集中。这一关联是DCT在图像压缩中优于DFT的根本原因。

## 四、实验结论

1. **数学本质**：DCT可视为对原信号进行偶对称延拓后执行DFT，由于对称性正弦项抵消，最终仅保留余弦分量。因此DCT是偶对称边界条件下的傅里叶分析特例。

2. **边界特性**：DFT隐含周期延拓假设，当信号首尾数值不等时会产生边界跳变，引入人为高频分量；DCT隐含偶对称延拓假设，边界自然连续光滑，无额外高频成分。

3. **能量集中性**：在Lena图像行信号（\(N=512\)）的实验中，DCT前51个系数占总能量的99.07%，而DFT仅为94.41%。DCT的能量集中性显著优于DFT，其根本原因在于偶对称延拓避免了边界突变。

4. **应用价值**：DCT具有实数运算、能量集中、边界连续等优点，因此在JPEG、H.264等图像/视频压缩标准中被广泛采用，成为图像处理领域比DFT更常用的变换工具。
