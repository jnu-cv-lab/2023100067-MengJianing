# 📄 实验报告：基于 FFT 与梯度方法的图像频率估计一致性分析

---

## 一、实验目的

1. 掌握基于 **傅里叶变换（FFT）** 的频率分析方法
2. 理解 **空间域梯度** 与频率之间的关系
3. 对比两种方法对图像局部频率的估计能力
4. 分析它们之间的一致性与差异

---

## 二、实验原理

### 2.1 FFT 频率分析方法

对图像块 ( I(x,y) ) 进行二维傅里叶变换：

[
F(u,v) = \mathcal{F}{I(x,y)}
]

功率谱定义为：

[
P(u,v) = |F(u,v)|^2
]

将频谱中心化后，定义径向频率：

[
r = \sqrt{u^2 + v^2}
]

对频率按半径排序，计算累积能量：

[
E(r) = \frac{\sum_{r_i \le r} P_i}{\sum P_i}
]

定义**95% 能量截止频率** ( f_{cut} )：

[
E(f_{cut}) = 0.95
]

 含义：频谱中95%的能量集中在该频率以内

---

### 2.2 梯度 RMS 频率方法

图像梯度定义：

[
\nabla I = \left(\frac{\partial I}{\partial x}, \frac{\partial I}{\partial y}\right)
]

梯度能量：

[
E[|\nabla I|^2]
]

根据频率域理论，有关系：

[
E[|\nabla I|^2] = 4\pi^2 \cdot \mathrm{Var}(I) \cdot f_{rms}^2
]

因此：

[
f_{rms} = \sqrt{\frac{E[|\nabla I|^2]}{4\pi^2 \cdot \mathrm{Var}(I)}}
]

 含义：频率的“能量加权平均”

---

### 2.3 两种方法区别

| 方法           | 含义             |
| ------------ | -------------- |
| FFT 95%      | 频率“上界”（能量覆盖范围） |
| Gradient RMS | 频率“中心”（平均能量位置） |

 因此二者**不完全等价**，但可能具有相关性

---

## 三、实验步骤

1. 读取灰度图像
2. 按 (64 \times 64) 分块
3. 对每个块执行：

   * FFT → 计算 95% 能量截止频率
   * 梯度 → 计算 RMS 频率
4. 去除平坦块（无有效频率）
5. 统计分析：

   * 相关系数（Pearson）
   * 线性拟合
   * 决定系数 (R^2)
6. 可视化：

   * 散点图（FFT vs Gradient）
   * 功率谱示意图

## 五、结果分析

### 5.1 相关性分析

* 当 ( r > 0.7 )：
  → 两种方法具有较强一致性
* 当 ( r \approx 0.4\sim0.7 )：
  → 中等相关，可用于粗略估计
* 当 ( r < 0.4 )：
  → 一致性较差

---

### 5.2 斜率分析

线性模型：

[
f_{grad} = k \cdot f_{fft} + b
]

* ( k < 1 )：梯度方法低估频率
* ( k > 1 )：梯度方法高估频率
* ( k \approx 1 )：尺度一致

---

### 5.3 偏差来源分析

#### （1）定义不同

* FFT → 最大频率范围
* RMS → 平均频率

 本质不同 → 不完全一致

---

#### （2）梯度对噪声敏感

* 梯度会放大高频噪声
* 导致频率偏高或波动

---

#### （3）FFT 截止是“硬边界”

* 对尾部频率敏感

#### （4）RMS 是“统计量”

* 对整体分布敏感

---

## 六、实验结论

1. FFT 95% 截止频率与梯度 RMS 频率之间通常存在**正相关关系**
2. 梯度方法可以作为一种**快速近似频率估计方法**
3. 但两者由于定义不同，无法完全等价
4. 梯度方法在存在噪声时误差较大
5. FFT 方法更精确，但计算复杂度更高

---

## 七、改进方向

1. 在梯度计算前加入高斯滤波（降低噪声）
2. 使用不同能量阈值（如 90%、99%）对比
3. 分析不同纹理类型（平滑/高频）
4. 使用 log-log 坐标增强分析
